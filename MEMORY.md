## 1. What this project is
Build a **hybrid** ESM-ready land component from LPJmL-FIT (LPJmL 5.6.004 + FIT, "waldspektrum" params). **[VERIFIED] Real source tree = `/home/jamirp/lpjml56fit`** (NOT `/home/jamirp/waldspektrum` — that path does not exist; git `b2e5ca9`, binary already built). - **S (slow, ML):** emulate per-cell trait/size **distributions** of trees (annual).
- **F (fast, physical, kept):** LPJmL-FIT daily biophysics (photosynthesis, water, soil thermal) — conserving.

## 2. Load-bearing VERIFIED facts (don't re-derive; re-confirm once in DESIGN)
- **[VERIFIED] Integration timestep = 1 day.** Loop: year→month→day→`update_daily`. No sub-daily physics except a numerical substep in the soil heat solver (`src/soil/apply_heatconduction_of_a_day.c`), not driven by sub-daily forcing.
- **[VERIFIED] Daily output = runtime config flag, no recompile.** `"timestep":"daily"` on an output → `config->withdailyoutput` auto-set (`src/lpj/fscanoutput.c`), daily write gated in `src/lpj/iterateyear.c`, per-variable selection in `iswrite2` (`src/lpj/fwriteoutput.c`). Only `VEGC,VEGN,GLOBALFLUX,CFTFRAC,CFTFRAC2,SDATE,SDATE2` capped at annual (`src/lpj/getmintimestep.c`); use `vegc_avg`/`agb` for sub-annual veg C.
- **[VERIFIED] No surface energy balance.** No `sensible`/`skin_temp`/`netrad`/`bowen`/`energy_balance` in core `src/`; no LE/H/Rn in `par/outputvars.js`. ET = Priestley–Taylor equilibrium / demand–supply (`src/numeric/petpar2.c`, `src/lpj/water_stressed.c`). Soil temp uses **air temperature as Dirichlet top BC** (`src/soil/update_soil_thermal_state.c`); no skin temperature. → H, G-as-flux, T_skin, Rn-closure must be **added** (component E), validated out-of-model.
- **[VERIFIED] Forcing consumed:** tas, precip, swdown, **net** longwave (`lwnet`), specific humidity `huss` (→VPD, **hard dependency** of FIT tree water stress via `getvpd.c`/`waterstress_tree.c`), CO₂, tamp(aux). **Wind read but not used** (SPITFIRE/N off). **Surface pressure not a forcing** (hard-coded `p=1.0e5` in `photosynthesis.c`). → E needs **wind + psurf** (new inputs).
- **[VERIFIED] State vector:** veg 7 C-pools/tree (leaf, sapwood, heartwood, root, sapwood_bg, heartwood_bg, debt); SOM `pool[LASTLAYER=22]{fast,slow}`; litter per PFT; soil water `w[23]`,`w_fw[23]`,`w_evap`,`rw_buffer`; ice `ice_depth/ice_fw/freeze_depth[23]`; thermal `enth[NHEATGRIDP]`(fundamental; NHEATGRIDP=NSOILLAYER·GPLHEAT=23 only because GPLHEAT=1 — key emulator to NHEATGRIDP); snow `snowpack`. Cell `Climbuf` = 20-yr running climate memory (main multi-year autocorrelation source → S conditioning). NSOILLAYER=23.
- **[VERIFIED] Fire is ON (GlobFIRM):** carbon closure MUST include `firec` (fire emission) and `flux_estabc` (establishment influx): `ΔC = NPP − Rh − firec + flux_estabc`, `NBP_atm = Rh + firec − NPP − flux_estabc`. Fire-free NEE will not close. Mortality drivers (S conditioning): water stress, **temp stress**, **growth efficiency (bm_inc history)**, age (`src/tree/mortality_tree_ind.c`). Source-file pointers: `getvpd.c`∈`src/spitfire/`, `waterstress_tree.c`∈`src/tree/`, `water_stressed.c`∈`src/lpj/`.
- **[VERIFIED+USER] Constant-CO₂ regime.** `with_nitrogen="no"` (verified) → no N limitation → CO₂ fertilization unbounded → LPJmL-FIT is run with **CO₂ held constant for the future** (user-confirmed) to avoid carbon runaway. So: CO₂ is a near-constant driver; **OOD test = warming/precip at constant CO₂, not rising CO₂**; SpeedyWeather's lack of a carbon cycle is a **non-issue** (NEE diagnostic-only); emulator is **not valid for CO₂-fertilization projections** (inherited limitation).
- **[VERIFIED] Allometry (re-derive diagnostics, don't co-predict):** height=k_latosa·Csap/(Cleaf·SLA·wooddens); crownarea Reinicke; LAI=Cleaf·SLA/crownarea; FPC=crownarea·nind·(1−e^(−k·LAI)); AGB=leaf+heart+sap.
- **[VERIFIED] Patch identity:** no explicit ID, but `(cell, patch-index)` is a **stable positional key** across years in the active `landuse="no"` config (the `ind` output emits it); per-tree `index` persistent while alive. → autoregressive lagged-state is feasible at patch level. Holds only while land-use/patch-merge/reservoir-stand paths stay inactive.
